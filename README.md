# GenomicsBench

A benchmark suite for genomics

### Python setup

```bash
python3.7 -m venv venv3
source venv3/bin/activate
pip install --upgrade pip
pip install -r requirements.txt
```
### Citation

If you use GenomicsBench, please cite this work:
